💣 Minesweeper Pro – Сапёр с разными размерами поля на 7 языках
Классическая игра «Сапёр» в терминале с гибкой настройкой размера поля, количества мин и цветным интерфейсом.
Открывай клетки, ставь флаги, не подорвись на мине!
Реализована на 7 языках программирования с единым принципом управления и расширенными возможностями.

🚀 Возможности
Выбор размера поля – от 9×9 до 30×16 (стандартные уровни easy, medium, hard).

Настройка количества мин – автоматически подбирается или задаётся вручную.

Цветной интерфейс – числа, флаги, мины и границы выделены разными цветами.

Управление – ввод координат и действия: открыть (Enter или пробел), поставить/снять флаг (F).

Рекурсивное открытие – автоматическое открытие пустых областей.

Счётчик мин – показывает оставшееся количество мин.

Таймер – отслеживает время игры (опционально).

Завершение игры – при победе или поражении выводится сообщение и результат.

Рекорды – сохраняются лучшие результаты (время, ходы) для каждого уровня (опционально).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [размер] [количество_мин]
Аргументы
размер – может быть предустановленным уровнем (easy, medium, hard) или числом для квадратного поля (например, 10 для 10×10). Если не указан – easy.

количество_мин – число мин (необязательно; для уровней задаётся автоматически: easy – 10, medium – 40, hard – 99).

Примеры
bash
# Запустить лёгкий уровень (9×9, 10 мин)
python minesweeper.py easy

# Средний уровень (16×16, 40 мин)
python minesweeper.py medium

# Сложный уровень (30×16, 99 мин)
python minesweeper.py hard

# Поле 20×20 с 50 минами
python minesweeper.py 20 50
Управление
Ввод координат в формате строка столбец (например, 3 5) для открытия клетки.

Ввод f строка столбец (например, f 3 5) для установки/снятия флага.

q – выход из игры.

🧠 Логика игры
Генерация поля – создаётся матрица с минами, числами и скрытыми клетками.

Открытие клетки – если под ней мина – игра заканчивается; если число – показывает его; если пусто – рекурсивно открываются соседние.

Флаги – игрок может ставить флаги на предполагаемые мины для защиты.

Победа – когда открыты все клетки, кроме заминированных, игра завершается победой.

Поражение – при открытии мины игра заканчивается, показывается расположение всех мин.

✨ Дополнительные фичи
Автоматическая адаптация – количество мин для уровней задаётся по умолчанию, но можно переопределить.

Цветной вывод – числа от 1 до 8 окрашены в разные цвета для лучшей читаемости.

Сохранение рекордов – лучшее время для каждого уровня сохраняется в файл (в большинстве реализаций).

Подсветка курсора – в некоторых версиях реализовано перемещение курсора (но для простоты используется ввод координат).
